# SQLAllocEnv (Visual FoxPro ODBC Driver)
> [!NOTE]  
>  This topic contains Visual FoxPro ODBC Driver-specific information. For general information about this function, see the appropriate topic under [ODBC API Reference](../content/ODBC-API-Reference.md).  
  
 Support: Full  
  
 ODBC API Conformance: Core Level  
  
 Allocates memory for an environment handle, *henv*,and initializes the ODBC call level interface for use by an application.  
  
 For more information, see [SQLAllocEnv](../content/SQLAllocEnv-Function.md) in the *ODBC Programmer's Reference*.
